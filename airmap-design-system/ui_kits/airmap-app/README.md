# Airmap App — UI Kit

A hi-fi recreation of the core Airmap workspace: left workspace sidebar, center Map canvas, right context panel.

## Components
- `Sidebar.jsx` — collapsible left navigation with workspaces, maps, deliverables
- `AppBar.jsx` — top bar with breadcrumb, search, user
- `Composer.jsx` — inline AI prompt with thread history
- `MapNode.jsx` — a single captured thought / reasoning node
- `ContextPanel.jsx` — right-side panel with promotions, versions, activity
- `DeliverableCard.jsx` — promoted deliverable preview
